  1. QUICK START GUIDE
================================================================

  When you plug in the drive, you'll see these launchers:

  WINDOWS USERS:  Double-click  WindowsLaunch.bat
  MAC USERS:      Double-click  MacLaunch.command

  You'll see a menu with four options:

    [1]  Survival AI + Knowledge Base  (RECOMMENDED)
    [2]  Survival AI Terminal Only
    [3]  Offline Medical Library
    [4]  Launch Everything

  Pick one and go.


  -- OPTION 1: SURVIVAL AI + KNOWLEDGE BASE (RECOMMENDED) --

  This is the flagship mode. A standalone app launches with a
  built-in interface that searches the US Army Survival Manual
  and Where There Is No Doctor in real time. When you ask a
  question, the system finds the most relevant passages and
  feeds them to the AI so it gives answers grounded in real
  survival and medical references — with source citations.

  The app opens as a native window — no browser needed.
  Wait 30-60 seconds for the AI to load, then start typing.


  -- OPTION 2: SURVIVAL AI TERMINAL ONLY --

  A lightweight text-only mode. The AI runs directly in your
  Terminal window without a browser. It's still a survival
  expert, but without the real-time reference search.

  Wait for the > prompt, then type your question.


  -- OPTION 3: OFFLINE MEDICAL LIBRARY --

  Opens a full medical encyclopedia in your web browser.
  80,000+ medical articles with diagrams, images, and
  treatment protocols — all running locally from the drive.
  No internet needed.

  The library runs at http://localhost:8888
  Use the search bar to find any topic.


  -- OPTION 4: LAUNCH EVERYTHING --

  Starts both the AI Knowledge Base AND the Medical Library
  side-by-side. Maximum knowledge access.


  -- NO BROWSER? USE THE TERMINAL --

  If you don't have a web browser, you can still access the
  medical encyclopedia from the command line:

  Mac:
    ./Offline\ Library/kiwix-search \
      ./Offline\ Library/library_data/*.zim "your search"

  Windows:
    "Offline Library\kiwix-search.exe" ^
      "Offline Library\library_data\*.zim" "your search"

  This prints matching article titles. You also have the
  Quick Reference folder with plain text survival guides
  that work in any text editor.


  HOW TO EXIT:

  - AI App: Close the app window, or press Ctrl+C in Terminal
  - AI Terminal: Press Ctrl+C twice, or type /bye
  - Library: Press Ctrl+C in the Terminal window
  - All session data is automatically destroyed.
    Nothing is saved. Unplug when done.
